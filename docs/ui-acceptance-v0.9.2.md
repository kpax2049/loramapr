# UI Acceptance Checklist - v0.9.2

Date: 2026-02-11

## Scope

- Layout and UX compactness updates.
- Sidebar navigation and map-first behavior.
- Debug isolation and reduced control clutter.

## Checklist

- [x] Left sidebar supports collapse/expand and drag resize.
- [x] Sidebar width/collapsed state persists across reloads.
- [x] Sidebar header/footer stay fixed while body content scrolls.
- [x] Sidebar tabs render and switch between Device/Sessions/Playback/Coverage/Debug.
- [x] Debug content is isolated under Debug tab and lazy-mounted.
- [x] Zen mode collapses the sidebar and preserves core map context.
- [x] Compact status strip remains visible on map.
- [x] Status strip is offset to avoid overlapping Leaflet zoom controls.
- [x] Map reflows correctly after sidebar collapse/resize.
